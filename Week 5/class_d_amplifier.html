Class D Amplifier

Class D amplifiers work fundamentally differently from linear amplifiers (Class A, B, AB). Instead of controlling how much a transistor conducts, Class D switches transistors fully ON or fully OFF at high frequency. This makes them extremely efficient — typically 90–95%. We'll build up the concept step by step: how the PWM signal is generated, what it looks like, and how a low-pass filter recovers the original audio.

Part 01

The Input Signal

We start with an analog input signal — a sine wave representing audio or a sensor reading.

In a linear amplifier, this signal directly controls how much current flows through the transistor. In Class D, we'll use it differently — as a reference for switching.
Part 02

The Comparator — Generating PWM

The core of a Class D amplifier is a comparator. It compares the input signal against a high-frequency triangle wave (the carrier). When the input is above the carrier, the output goes HIGH. When below, it goes LOW. This creates a pulse-width modulated (PWM) signal.

Notice how the duty cycle (proportion of time spent HIGH) varies with the input signal. When the input is at its peak, the PWM is HIGH most of the time. When at its trough, it's LOW most of the time. At zero, it's a 50% duty cycle.
Duty Cycle = (Vin + Vcarrier,pk) / (2 × Vcarrier,pk)
Part 03

The Switching Stage — MOSFET Half-Bridge

The PWM signal drives a pair of complementary MOSFETs (NMOS and PMOS) arranged in a half-bridge. Unlike BJTs in linear amplifiers, these MOSFETs operate as digital switches — fully ON or fully OFF.

Because the MOSFETs are either fully on (RDS(on) ≈ milliohms, nearly zero voltage drop) or fully off (zero current), the power dissipated in the switches is almost zero. This is why Class D efficiency approaches 100%.
Pswitch ≈ 0   (VDS × ID ≈ 0 in both states)
Part 04

The LC Low-Pass Filter — Recovering the Signal

The PWM output is a high-frequency square wave that rapidly switches between VCC and GND. To recover the original analog signal, we pass it through an LC low-pass filter. The inductor resists rapid current changes, and the capacitor smooths the voltage. The filter's cutoff frequency is set above the signal bandwidth but well below the switching frequency.

The LC filter averages the PWM signal. During HIGH periods, current flows through the inductor into the capacitor and load. During LOW periods, the inductor's stored energy maintains the current. The result is a voltage proportional to the duty cycle — but not perfectly smooth. Try dragging the fsw/fsig slider above. At low ratios (5–10x), you can clearly see the switching ripple riding on top of the output. At high ratios (40x+), the ripple becomes negligible. A 2nd-order LC filter attenuates ripple by the ratio squared (−40 dB/decade).
fcutoff = 1 / (2π√(LC))   where   fsignal << fcutoff << fswitching

Part 06

Why MOSFETs?

Class D amplifiers require transistors that can switch very fast with minimal loss. MOSFETs are the clear choice over BJTs for several reasons:

  • Switching speed: BJTs suffer from storage time — minority carriers in the base must be swept out before the transistor turns off. MOSFETs are majority-carrier devices with no storage time, so they switch much faster.
  • Voltage-driven gate: BJTs need continuous base current to stay on. MOSFETs are voltage-driven — once the gate capacitance is charged, zero steady-state current flows into the gate. This simplifies the gate driver circuit.
  • Lower on-resistance: Modern power MOSFETs achieve RDS(on) in the low milliohm range, producing less voltage drop than a BJT's VCE(sat) at high currents.
  • Modern applications: Class D is used in audio amplifiers, motor drivers (H-bridges), DC-DC power converters, and biomedical devices wherever efficiency matters.
Class D is now the dominant amplifier topology for portable and battery-powered biomedical devices, where power efficiency directly impacts battery life and thermal management.
