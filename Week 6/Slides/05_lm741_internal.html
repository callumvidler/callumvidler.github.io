Internal Circuitry of the LM741 Op-Amp
  • Detailed transistor-level analysis is outside the scope of this topic
  • The labelled regions identify the principal functional blocks
  • The complete schematic provides context for the simplified op-amp model
Differential inverting stage Q1–Q7 · lateral PNP input pair Amplifies the voltage difference between the two inputs while rejecting any common-mode signal. Sets the op-amp's high input impedance and low input bias current.
Current mirrors Q8/Q9 · Q10/Q11 · Q12/Q13 Internal bias network that copies one reference current to every stage, giving each transistor a stable operating point and acting as a high-impedance active load to maximise gain.
Class A gain stage Q15, Q19, Q22 A single high-gain common-emitter transistor biased in class A that provides the bulk of the op-amp's open-loop voltage gain (~100 dB). The 30 pF Miller cap sets the dominant pole for stability.
Voltage level shifter Q16 + 4.5 kΩ / 7.5 kΩ (VBE multiplier) A VBE-multiplier that holds a fixed DC offset between the two output transistors, pre-biasing them just into conduction so they switch smoothly and avoid class-B crossover distortion.
Output stage Q14, Q17, Q20 · class-AB push-pull A complementary push-pull pair that sources or sinks load current at the output pin, giving the op-amp a low output impedance and a rail-to-rail-ish voltage swing.
